## Realm转发脚本使用
### 功能
- 一键添加配置
- 一键安装、更新、删除Realm
- 支持安装包本地安装
- 查看存在的配置规则
- 支持tls,ws,wss加密
- 支持证书配置
- 启动与查看日志
### 使用
``` bash
wget -N https://raw.githubusercontent.com/kentinya/Script-repository/main/Realm/realm.sh && chmod +x realm.sh
```
``` bash
bash realm.sh
```
